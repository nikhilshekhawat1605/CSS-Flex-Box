# CSS-Flex-Box

**Learning and practicing CSS Flexbox with real layout examples.**

📦 Flexbox Properties  
🔹 flex-direction

Defines direction of flex items:

row

row-reverse

column

column-reverse

🔹 justify-content

Aligns items horizontally (main axis):

flex-start

flex-end

center

space-between

space-around

space-evenly

🔹 align-items

Aligns items vertically (cross axis):

flex-start

flex-end

center

stretch

baseline

🔹 flex-wrap

Controls wrapping of items:

nowrap

wrap

wrap-reverse

🔹 order

Controls item order:

order: number; (default is 0)

🔹 flex-grow

Controls how much item grows:

flex-grow: number; (default is 0)
